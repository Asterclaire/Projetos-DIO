HTML

A tag <html> define o início e o fim de um documento HTML.

HEAD

A tag <head> contém informações sobre o documento, como título, metadados, links para estilos e scripts.

TITLE

A tag <title> define o título do documento, que é exibido na barra de título do navegador ou na guia.

BODY

A tag <body> contém todo o conteúdo visível do documento, como texto, imagens, links, etc.

H1

A tag <h1> define um cabeçalho de nível 1, geralmente usado para o título principal da página.

P

A tag <p> define um parágrafo de texto.

A

A tag <a> define um hiperlink, que leva o usuário para outra página da web quando clicado.

IMG

A tag <img> define uma imagem a ser incorporada na página.

LINK

A tag <link> define a relação entre o documento atual e um recurso externo, como uma folha de estilo CSS.

TEXT

A tag <input type="text"> define um campo de entrada de texto em um formulário.

FORM

A tag <form> define um formulário HTML usado para coletar dados do usuário. Pode conter vários elementos de entrada, como campos de texto, botões de opção, botões de seleção, entre outros.
